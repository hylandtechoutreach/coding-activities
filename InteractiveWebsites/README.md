# Interactive Websites with JavaScript: Lesson Plan
This lesson introduces JavaScript with an interactive presentation, coding activity, and formative assessment. Experience with HTML and CSS is not required, but this lesson can be paired with the [Building Websites](BuildingWebsites/) activity if desired. By the end of the lesson, students should have a quiz about themselves, first [building one from scratch](QuizAboutMeCodeAlong.md), and then building one [based on a template](TemplateProject.md).

### Agenda

| Item | Time |
|-|-|
| Introduction | 10m |
| Code-Along | 30m |
| Template Quiz Project | 10m |
| Formative Assessment | 10m |

## Introduction
Present the [introductory slides](https://docs.google.com/presentation/d/1sj7AFa9oXMKHpTl8uK4RefScuOFcadkRpHjKkxkdlkc/edit?usp=sharing) that go over some of the basics of JavaScript. They cover quite a lot, but it's more about vague familiarity than actual retention.

There are several areas where students can play around with the code - ask them to come up to the instructor computer to see how they do, or ask them for suggestions about what to type. They are not expected to visit the example projects on their own computers.

Additionally, there is a [presentation page](JavaScriptPresentation.md) with links to all of the example projects for reference.

## Code-Along: Quiz About Me
Follow the [code-along instructions](QuizAboutMeCodeAlong.md) to help the students build up a quiz about themselves!

They are expected to start from [this template](https://hytop.onrender.com/e/www-js), but as long as the students have the **script.js** file properly linked from **index.html**, and a stub `startQuiz` function definition, they will be good to go.

## Template Project: Fancy Quiz
Follow the [template project](TemplateProject.md) instructions to show students a different approach to quiz building. Starting from the [example template](https://hytop.onrender.com/e/quiz-about-me), all they have to do is change the questions, answers, and variables in the **questions.js** file to make the quiz their own.

This isn't as rewarding as coding from scratch, but it should hopefully be fun.

## Formative Assessment: Gimkit
At the end of the session, play the [Gimkit](https://www.gimkit.com/view/6a03762298529dd14140ffbd).

Hopefully the students will remember something.
