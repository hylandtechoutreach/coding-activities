# 3D Worlds with A-Frame: Lesson Plan
In this activity, students will create a 3D world using [A-Frame](https://aframe.io/).

[Click here for the code-along instructions.](WorldbuildingCodeAlong.md)

## Sharing
Students can share their work - all they need to know is their project name! Append it to the end of `hytop.onrender.com/pf/` to create a view URL. It is also possible for them to change their project name in HyTOP, and add a description - this can help them recover their work if needed, and may make it easier to find.

## STEAM Event Groups
Each event group can fork a different starter project to ensure they are separately viewable. [short.hyland.com/afsA](https://short.hyland.com/afsA) should go to the [afsa](https://hytop.onrender.com/e/afsa) project, and [short.hyland.com/afsB](https://short.hyland.com/afsB) should go to the [afsb](https://hytop.onrender.com/e/afsb) project. Then, all of the student projects should automatically be viewable from the forks pages - group A on the [forks of afsa page](https://hytop.onrender.com/forks-of/afsa), and group B on the [forks of afsb page](https://hytop.onrender.com/forks-of/afsb).
