# Processing 2D Scene Challenges
Here are some ideas for challenges that could make your scene even better!

- Draw a cloud
- Make the sun move across the sky
- Make the scene bigger
- Add an image of a bird
- Add an ocean behind the ground
- Change the scene to nighttime
  - Add stars
  - Try to make a crescent moon
- Make it winter
  - Add snow
  - Draw a snowman
- Change the scene to a beach at sunset
- Draw a gray mountain
