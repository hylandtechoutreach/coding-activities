# Building a 2D Scene in p5.js: Lesson Plan
This lesson introduces [Processing](https://processing.org), and specifically [p5.js](https://p5js.org), as a platform for creative coding. By the end of the activity, students will have built their own small scene, and have some familiarity with Processing.

### Agenda

| Item | Time |
|-|-|
| Presentation | 20m |
| Code-Along | 30m |
| Formative Assessment | 10m |

## Introduction
Present the [introductory slides](https://docs.google.com/presentation/d/1qI1nRh0afxfExw-63fHfYYUENfr_1iH_l5vfA4gFNW0/edit?usp=sharing) that go over some of the basics of Processing and p5.js. The presentation has a section where students are meant to search for examples in OpenProcessing to see some of what's possible. There are also a couple of games built into the presentation. These are all linked from the [homepage](StudentDesc.md).

There may be some examples where the students can adjust the code, or make suggestions.

Additionally, there is a [presentation page](ProcessingPresentation.md) with links to all of the example projects for reference.

## Code-Along: Building a 2D Scene
Follow the [code-along instructions](CodeAlong2dScene.md) to help the students build up a two dimensional scene!

They should start from the [starter code](https://hytop.onrender.com/e/start-p5-scene).

## Formative Assessment: Blooket
At the end of the lesson, play the [Blooket]().

Hopefully the students will remember something.
