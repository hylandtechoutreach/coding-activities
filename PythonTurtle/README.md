# Turtle Graphics with Python
This lesson introduces the Python programming language using turtle graphics. By the end of the activity, students will have coded their own turtles that draw their own shapes with whatever colors they'd like!

### Agenda: 60 minutes

| Item | Time |
|-|-|
| Opening Slides | 5m |
| Code-Along | 30m |
| Self-Paced Time | 10m |
| Closing Slides | 5m |
| Blooket | 10m |

## Opening Slides
Present the [opening slides](https://docs.google.com/presentation/d/1PAQZPZsGwpov2sZwXmCEw3hhAC_LiA6IjR34VSD93TM/edit?usp=sharing) to introduce Python.

## Code-Along
Walk through the [code-along instructions](PythonTurtleCodingActivity.md) with the students. Make sure to ask for suggestions about what to type.

## Self-Paced Time
After the code-along, students can continue working on their Turtle projects. 

### Examples
Students can view the [examples](TurtleExamples.md) of what Turtles can do.

### Challenges
Students can try to complete various [challenges](TurtleChallenges.md)
