# Platformer Hacker with JavaScript: Lesson Plan
This lesson tasks students with playing and hacking a [game](https://hytop-backend.onrender.com/static/hackyformer/index.html?sessionName=default) to try to get a high score! **Variables** are fundamental computer science concept covered by this activity.

It is designed to be offered for anywhere from 5-15 minutes, but can be extended if students really want to achieve the highest score possible.

## Procedure
To facilitate the activity, follow these steps:

1. Have the students visit the coding activity homepage at [short.hyland.com/coding](https://short.hyland.com/coding)
1. Have them click the [link](https://hytop-backend.onrender.com/static/hackyformer/index.html?sessionName=ca1) to get started
1. If needed, tell everybody to go to a specific session based on name  
  - If there are already high scores loaded, change the session name
1. Try to see if students can figure out how to play on their own
1. If students struggle, explain the controls and objectives to the group
1. Help students individually
1. If anyone wants to make the game easier, or wants to reach the coins at the final level - use that as an opportunity to start explaining the hacking of variables!
1. Explain that **variables** in computer science are named containers for data that can change. For example, changing the number after `JUMP_SPEED=` can make the player jump higher or lower
1. Continue playing the game and see how high the scores can get!
