# Retro Games: Lesson Plan
This hour-long activity introduces [Microsoft MakeCode Arcade](https://arcade.makecode.com/), and shows students how they can use the platform to create a retro-style [platformer](https://en.wikipedia.org/wiki/Platformer) game.

### Agenda
Here is a rough outline of the activity as a whole:

| Item | Time |
|-|-|
| Slides | 10m |
| Follow-Along | 40m |
| Blooket | 10m |

## Slides
To kick things off, present the [Opening Slides](https://docs.google.com/presentation/d/1i82MDaP1AIOK996pfpX4fGhrbqDGnJpcZxKRadNFz5M/edit?usp=sharing). The last slide introduces the Follow-Along activity, and provides a link for the students to get started: **[short.hyland.com/mca](https://short.hyland.com/mca)** _(that link should redirect to [https://arcade.makecode.com/#editor](https://arcade.makecode.com/#editor) when visited)_

## Follow-Along: Simple Platformer
Follow [the instructions](FollowAlong.md) to help students start building a retro platformer game of their own!

## Blooket
End the session by playing the [Blooket](https://dashboard.blooket.com/set/65084d4418adbc094405e876).
